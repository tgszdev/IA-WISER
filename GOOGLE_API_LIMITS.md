## 📈 Dicas para Economizar Requisições

1. **Use Cache Local**
   - Salve respostas frequentes
   - Reutilize dados quando possível

2. **Batch Requests**
   - Agrupe múltiplas perguntas
   - Use o endpoint /api/batch-query

3. **Optimize Prompts**
   - Seja direto e conciso
   - Evite requisições desnecessárias
